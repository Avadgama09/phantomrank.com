# 🚀 PhantomRank

**PhantomRank** is an AI Visibility Tracking platform built for marketing agencies. It monitors how brands are cited and mentioned across major Large Language Models (LLMs) including ChatGPT, Perplexity, Gemini, and Claude.

## Key Features
- ✅ **AI Visibility Tracking**: Monitor brand mentions and citation frequency in real-time.
- ✅ **Competitive Benchmarking**: Track share of voice against industry competitors within AI search results.
- ✅ **Agency-First Dashboard**: Manage multiple client brands from a single intelligence layer.
- ✅ **AEO Optimization**: Insights designed to help content get picked up by generative engines.

## Tech Stack
- **Framework**: [Astro 5](https://astro.build/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Deployment**: [Vercel](https://vercel.com/)
- **Analytics**: Vercel Speed Insights & Web Analytics
